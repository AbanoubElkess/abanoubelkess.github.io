A collaboration with major semiconductor manufacturing clients to build automated data cleaning pipelines for CD-SEM (Critical Dimension Scanning Electron Microscope) metrology. The system filters out low-quality scans and classifies defect features.

### Key Highlights

- **Deep Learning Classifier**: Developed custom Convolutional Neural Networks (CNNs) to classify structural metrology errors and noise from client SEM runs.
- **QT-Based Dashboard**: Implemented a cross-platform desktop application using PyQt/PySide to visualize large-scale SEM datasets, inspect neural network segmentations, and audit annotations.
- **Yield Improvement**: Automated physical modeling file generation, resulting in a 30% reduction in manual verification overhead for RET/OPC modeling.
