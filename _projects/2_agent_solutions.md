An agentic framework designed to automate IC layout verification and pattern-less PDK (Process Design Kit) rule check synthesis. The system orchestrates multiple localized agents to run DRC (Design Rule Checking), inspect violations, and propose layout changes.

### Key Highlights

- **Automated PDK Verification**: Built intelligent agents that execute DRC/LVS decks, analyze compile logs, and synthesize test patterns to cover corner cases.
- **Multi-Agent Collaboration**: Leveraged LLMs acting as specialized workers (Developer, Compiler, QA) to resolve conflicting verification errors.
- **Performance**: Dramatically decreased setup times for verification runs and increased rule coverage by 25% through automated pattern generation.
